# 카드뉴스 자동화 도구 — 사용 설명서

글(기사·대본·메모)을 붙여넣으면 AI가 인스타그램 카드뉴스(1080×1350 PNG 세트)로 만들어주는 로컬 브라우저 에디터입니다. 이 문서는 실제로 사용하는 방법을 처음부터 끝까지 안내합니다.

---

## 1. 시작하기 전에 필요한 것

| 항목 | 필수 여부 | 설명 |
|---|---|---|
| Node.js | 필수 | [nodejs.org](https://nodejs.org)에서 설치. 서버(`server.js`)를 돌리는 데 필요합니다 |
| Claude Code CLI 로그인 | AI 변환에 필요 | `claude` 명령어가 터미널에서 실행돼야 "✨ AI로 변환"이 동작합니다. 없어도 "빠른변환"은 사용 가능 |
| Chrome 계열 브라우저 | 권장 | Edge, Chrome 등. PNG/ZIP 내보내기가 안정적으로 동작합니다 |
| (선택) Gemini / OpenAI / Pexels API 키 | 배경 이미지 자동 생성/검색에 필요 | 없어도 로컬 배경 풀 셔플, 업로드는 그대로 사용 가능 |

---

## 2. 실행 방법

1. `cardnews` 폴더 안의 **`start.bat`**을 더블클릭합니다.
2. 검은 창(서버 로그)이 뜨고, 몇 초 뒤 브라우저가 자동으로 열립니다.
3. 브라우저가 안 열리면 주소창에 `http://localhost:8787/`을 직접 입력하세요.
4. 서버 로그 창에는 지금 사용 가능한 배경 생성 수단(Codex/Gemini/OpenAI/Pexels)이 ✅/❌로 표시됩니다. 이 창은 작업 중 계속 켜둬야 합니다 (끄면 AI 변환·배경 생성·내보내기 일부가 멈춥니다).

> 같은 사무실(같은 Wi-Fi/공유기) 안의 팀원은 설치 없이 브라우저에서 `http://<호스트 PC의 IP>:8787/`로 접속하면 됩니다. 호스트 PC의 IP는 서버 로그 창에 같이 출력됩니다.

---

## 3. 화면 구성

에디터는 3단으로 나뉩니다.

- **왼쪽 (필름스트립)**: 카드 목록 썸네일. 클릭해서 선택, 드래그해서 순서 변경, ✕로 삭제, "＋ 카드 추가"로 새 카드 추가. 맨 아래에는 기능 감지 배지와 "⚙ 서버 설정"이 있습니다.
- **가운데 (스테이지)**: 지금 선택된 카드의 실시간 미리보기. "✨ 글로 카드 만들기", "⬇ 전체 ZIP", "💾 JSON", "📂 불러오기", "🗑 초기화" 버튼이 여기 있습니다.
- **오른쪽 (필드 패널)**: 스타일 선택, 포인트 색, 배경 사진 관리, 현재 카드의 텍스트 필드 편집.

---

## 4. 카드뉴스 만드는 3가지 방법

### 방법 A. 글 하나로 AI 자동 생성 (추천)

1. 가운데 상단의 **"✨ 글로 카드 만들기"** 버튼을 누릅니다.
2. 텍스트 상자에 원본 글(블로그 초안, 기사, 대본 등)을 통째로 붙여넣습니다.
3. **"✨ AI로 변환"**을 누릅니다. 20~40초 정도 걸립니다.
4. 표지 → 도입 → 핵심 단계 → (있으면) 비교표 → (있으면) 인용 → 마무리 CTA 순서로 8~12장 정도의 카드 덱이 자동 생성됩니다.
5. 사진 스타일(시네마틱/매거진/만화)을 쓰고 있다면, 변환 직후 자동으로 ① 임시 배경이 랜덤으로 채워지고 ② 배경 생성 수단이 있으면 표지·인용·마무리 카드가 글 주제에 맞는 배경으로 순차 교체됩니다.
6. 결과가 마음에 안 들면 필드 패널에서 각 카드의 텍스트를 직접 수정하면 됩니다.

> AI 변환이 실패하면 "start.bat이 켜져 있나요?" 안내가 뜹니다. 서버 창이 꺼져 있거나 `claude` CLI 로그인이 안 되어 있는 경우가 대부분입니다.

### 방법 B. 빠른변환 (AI 없이, 오프라인)

같은 패널에서 **"빠른변환"**을 누르면 서버나 AI 없이 즉시 규칙 기반으로 카드가 만들어집니다.
- 글의 첫 줄 → 표지 제목
- `##` 또는 `1.` 로 시작하는 줄 → 각각 새 단계 카드
- `|`가 2개 이상 들어간 줄들 → 비교표 카드
- 결과 품질은 AI 변환보다 단순하지만, 서버 없이도 언제든 즉시 사용할 수 있습니다.

### 방법 C. 처음부터 직접 만들기

"＋ 카드 추가"로 카드를 하나씩 추가하고, 오른쪽 필드 패널에서 타입(표지/본문/인용/비교표/마무리)을 고른 뒤 필드를 채우면 됩니다.

**텍스트 입력 문법 (모든 텍스트 칸 공통)**
| 입력 | 결과 |
|---|---|
| `Enter` | 줄바꿈 |
| `*단어*` | 포인트 색으로 강조 |
| `**단어**` | 굵게 |

## 5. 스타일(디자인) 바꾸기

오른쪽 패널 맨 위 **"스타일 정체성"** 드롭다운에서 6가지 중 고를 수 있습니다.

| 스타일 | 느낌 | 사진 배경 |
|---|---|---|
| 시네마틱 | 다크 톤, 블루 포인트 | 필수 |
| 매거진 형광 | 깃발형 라벨, 노랑 형광펜 강조 | 필수 |
| 만화 오버레이 | 세로바 라벨, 텍스트 그림자 | 필수 |
| 픽셀 인포 | 굵은 헤드라인 폰트, 청키 배지 | 불필요 |
| 도들(손글씨) | 손글씨 폰트, 비정형 둥근 테두리 | 불필요 |
| 화이트 & 블루 | 깔끔한 화이트 배경 | 불필요 |

스타일을 바꾸면 폰트·라벨 모양·강조 방식이 세트로 함께 바뀝니다. **포인트 색**은 11색 팔레트나 커스텀 컬러피커로 스타일과 별개로 다시 조정할 수 있습니다.

배경 사진은 **카드마다, 스타일마다 따로 저장**됩니다. 예를 들어 시네마틱 스타일에서 올린 사진은 매거진 스타일로 바꿔도 섞이지 않고, 다시 시네마틱으로 돌아오면 그대로 남아있습니다.

---

## 6. 배경 사진 넣는 4가지 방법

사진이 필요한 스타일(시네마틱/매거진/만화)을 선택하면 필드 패널에 배경 관리 영역이 나타납니다.

1. **🎲 셔플**: "이 카드" 또는 "전체" 버튼으로 미리 준비된 배경 풀에서 무작위로 채워줍니다. 가장 빠른 방법입니다.
2. **직접 업로드**: 가진 사진 파일을 올립니다.
3. **무료 사진 검색**: 검색어(영어 권장)를 입력하면 Pexels(키가 있으면) 또는 Openverse(키 없이도 동작)에서 무료 사진을 찾아 클릭 한 번으로 적용합니다.
4. **AI 배경 생성**: 장면 설명을 영어로 입력하고 "✨ AI로 배경 생성"을 누르면 Codex(ChatGPT 구독) → Gemini 키 → OpenAI 키 순서로 시도해 배경을 만들어줍니다. 아무것도 설정되어 있지 않으면 안내 메시지가 뜹니다.

배경을 넣은 뒤에는 **"배경 하단 어둡기" 슬라이더**로 글자가 잘 보이도록 하단 그림자 농도를 카드마다 조절할 수 있습니다.

> API 키를 쓰려면 `tools/gemini-key.txt.example`, `tools/openai-key.txt.example`, `tools/pexels-key.txt.example` 파일을 복사해서 `.example`을 지우고 실제 키를 첫 줄에 붙여넣으세요. 서버를 껐다 켜지 않아도 저장하는 즉시 반영됩니다.

## 7. 완성한 카드뉴스 내보내기

- **⬇ 전체 ZIP**: 모든 카드를 1080×1350 PNG로 순서대로 캡처해서 `01.png ~ NN.png`로 압축 다운로드합니다. 인스타그램에 올릴 최종 결과물입니다.
- 개별 카드만 필요하면 필름스트립에서 원하는 카드를 선택한 뒤 스테이지 미리보기를 기준으로 별도 저장(다음 업데이트에서 개별 내보내기 버튼 추가 예정)하거나, ZIP 안에서 필요한 파일만 골라 쓰면 됩니다.

**주의**: 반드시 `start.bat`으로 서버를 켠 상태(`http://localhost:8787/...`)에서 내보내기를 해야 합니다. `index.html` 파일을 직접 더블클릭해서 여는 `file://` 방식으로는 브라우저 보안 정책 때문에 이미지 내보내기가 실패할 수 있습니다.

---

## 8. 작업 저장하고 이어서 하기

- 편집 중인 모든 내용은 **입력하는 즉시 브라우저에 자동 저장**됩니다(`localStorage`). 브라우저를 껐다 다시 열어도 이어서 작업할 수 있습니다. (단, 브라우저의 사이트 데이터를 지우면 함께 사라집니다.)
- **💾 JSON**: 지금 작업 중인 덱 전체를 파일로 내보냅니다. 백업하거나 다른 PC/팀원에게 전달할 때 사용하세요.
- **📂**: JSON 파일을 불러와 그대로 이어서 편집합니다.
- **🗑 초기화**: 저장된 내용을 전부 지우고 기본 예시 덱으로 되돌립니다. 되돌릴 수 없으니 신중하게 사용하세요.

---

## 9. 팀원과 함께 쓰기

1. 호스트 PC에서 `start.bat`을 켜둡니다.
2. 서버 로그 창에 뜨는 `http://192.168.x.x:8787` 같은 주소를 팀원에게 알려줍니다.
3. 팀원은 브라우저로 그 주소에 접속하기만 하면 설치 없이 바로 사용할 수 있습니다.
4. 만약 접속 주소가 자동으로 안 맞으면, 필름스트립 왼쪽 아래 **"⚙ 서버 설정"**에서 서버 주소를 직접 입력하고 저장할 수 있습니다. 저장하면 바로 연결 상태(✅/⚠️)를 확인해줍니다.

## 10. 자주 겪는 문제

| 증상 | 원인 / 해결 |
|---|---|
| "AI로 변환"이 실패한다 | 서버 창(검은 창)이 꺼져 있거나, `claude` CLI가 로그인되어 있지 않습니다. 서버 창을 다시 켜거나 터미널에서 `claude` 로그인 상태를 확인하세요 |
| ZIP/PNG 내보내기가 안 된다 | `index.html`을 더블클릭해서 `file://`로 열었을 가능성이 큽니다. `start.bat`으로 켠 `http://localhost:8787/` 주소에서 다시 시도하세요 |
| 배경 검색/생성이 안 된다 | Pexels/Gemini/OpenAI 키가 설정되지 않았거나, Codex 로그인이 안 되어 있습니다. 필름스트립의 기능 배지(✅/⭘)로 지금 뭐가 켜져 있는지 확인할 수 있습니다. 키 없이도 🎲 셔플과 직접 업로드는 항상 됩니다 |
| 포트 8787이 이미 사용 중이라고 뜬다 | 이미 서버가 켜져 있는 상태입니다. 기존 서버 창을 찾아 그대로 쓰거나, 작업관리자에서 관련 node 프로세스를 종료한 뒤 다시 시도하세요 |
| 텍스트가 카드 밖으로 넘친다 | 자동으로 글자 크기가 줄어들도록 되어 있지만, 그래도 넘치면 본문을 조금 줄여서 다시 입력해주세요 |

## 11. 아직 없는 기능 (다음 업데이트 예정)

- 카드 1장만 골라서 PNG로 바로 내보내기 버튼
- 픽셀 스타일의 마스코트 이미지
- 배경 풀 사진 개수 확장 (현재 스타일당 4장 → 스펙상 9/6/6장)
